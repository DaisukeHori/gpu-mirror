# QA チェックリスト

リリース前に iOS / Android / Web で確認する項目。各項目を実機またはシミュレータで確認し、結果を記録する。

---

## 認証フロー

- [ ] SSO ログイン → セッション確立 → ホーム画面に遷移する
- [ ] ログアウト → ログイン画面に戻る
- [ ] 未認証で `/` にアクセス → ログインに自動リダイレクトされる
- [ ] (Android) アプリキル後にカスタムスキームで起動 → セッション復帰する

---

## 利用規約

- [ ] 利用規約が表示される
- [ ] 最後までスクロールしないとチェックボックスが有効にならない
- [ ] (大画面) コンテンツが 1 画面に収まる場合でもチェックボックスが有効になる
- [ ] 同意して「次へ」→ 写真の準備画面に遷移する
- [ ] **初回同意後**、同じ端末でホーム「はじめる」→ **規約を省略して** 写真の準備へ直行する（再同意が必要なときは `apps/mobile/lib/terms-consent.ts` の `TERMS_CONSENT_VERSION` を上げる運用）

---

## 写真の準備・カメラ

- [ ] 撮影ガイドが表示される
- [ ] カメラ権限ダイアログが表示される（初回）
- [ ] カメラで撮影 → プレビューが表示される
- [ ] 「撮り直す」→ カメラに戻る
- [ ] 「この写真を使う」→ セッション作成 + アップロード → スタイル選択に遷移
- [ ] ライブラリ選択ボタン → フォトライブラリ権限ダイアログ（初回）
- [ ] ライブラリから写真を選択 → アップロード → スタイル選択に遷移
- [ ] (Android) 権限拒否時 → アラート表示

---

## スタイル選択 (Explore)

- [ ] Pinterest タブ: 検索バーが動作する
- [ ] Pinterest タブ: 長押しで画像を選択できる
- [ ] Pinterest タブ: (Android) 戻るボタンで WebView 内を戻る
- [ ] Pinterest タブ: ピン詳細画面で「この画像を追加」ボタンが動作する
- [ ] カタログタブ: 一覧が表示される
- [ ] アップロードタブ: ライブラリから画像を選択できる
- [ ] カラータブ: カラーパレットが表示・選択できる
- [ ] スタイルトレイに選択したスタイルが追加される
- [ ] 「生成へ進む」→ 確認画面に遷移する

---

## 確認画面

- [ ] 選択したスタイルがカードグリッドで表示される
- [ ] カラーの割当・変更ができる
- [ ] 「新しいスタイルを生成」→ 生成画面に遷移する

---

## 生成中

- [ ] プログレスボードが表示される
- [ ] SSE でリアルタイム進捗が更新される
- [ ] 全て完了 → 自動で結果画面に遷移する
- [ ] 途中で Exit → 確認ダイアログ → ホームに戻る

---

## 結果画面

- [ ] 映えタイルが表示される
- [ ] タイルタップ → 横スワイプビューアが開く
- [ ] お気に入りトグルが動作する（API 反映も確認）
- [ ] 「共有シートを開く」→ 共有ダイアログが表示される
- [ ] (キャッシュ済み file:// 画像) 共有がダウンロードなしで動作する
- [ ] フリーワード入力 → 「再生成」→ 生成画面に遷移する
- [ ] 「スタイル追加」→ スタイル選択画面に遷移する

---

## 管理画面 (Admin)

- [ ] 設定画面 → 「施術ログを開く」→ セッション一覧が表示される
- [ ] 検索・フィルタが動作する
- [ ] セッション詳細が表示される
- [ ] CSV エクスポート → ファイルが共有される
- [ ] (Web) CSV エクスポート → ファイルがダウンロードされる

---

## プラットフォーム固有

### iOS
- [ ] iPad シミュレータでレイアウトが崩れない
- [ ] ステータスバー・ノッチ周りの余白が適切

### Android
- [ ] エミュレータ / 実機でアプリが起動する
- [ ] ステータスバーの高さに応じたヘッダー余白が適切
- [ ] `adb reverse tcp:3000 tcp:3000` でローカル API に接続できる
- [ ] システム戻るボタンの動作が適切

### Web (Expo Web)
- [ ] ブラウザでアプリが表示される
- [ ] Pinterest タブがカタログタブにフォールバックする
- [ ] 共有が Web Share API / クリップボードで動作する

---

## エッジケース

- [ ] 生成失敗 → リトライボタン → 再生成が成功する
- [ ] セッション ID が不正なディープリンク → 無限スピナーにならない
- [ ] `sessionId` なし・欠落パラメータで探索/確認/生成/結果を開く → 「表示できません」等の案内とホームへ戻る動線になる
- [ ] ネットワーク切断中 → 適切なエラーメッセージが表示される
