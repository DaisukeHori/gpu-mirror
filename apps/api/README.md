# REVOL Mirror API

BFF (Backend for Frontend) API。Next.js App Router で構築。

詳細なセットアップ手順・環境変数・デプロイ方法はルートの [README.md](../../README.md) を参照してください。

## 開発

```bash
npm run dev     # Next.js 開発サーバー起動
npm test        # テスト実行
npm run lint    # ESLint 実行
npm run build   # プロダクションビルド
```

## API エンドポイント
